# Hello Universe

This project creates the text Hello Universe with style into your terminal. It prints the word "Hello" in cyan, and the word "Universe" in green.

## Instructions for Build and Use

Steps to build and/or run the software:

1. Download a Code Editor
2. In the console type: dotnet build
3. Then, in the console again, type: dotnet run

Instructions for using the software:

1. Watch as it loads
2. Read the masterpiece
3. Do 20 push-ups
